# Document Classifier (Python + OpenCV)

This project is an image processing tool designed to identify and classify the type of document presented in a photo. It uses computer vision to distinguish between various formats.

Developed in collaboration with **[JFuB](https://github.com/JFuB)**.

## 🚀 Features

The program analyzes a photo and classifies it into one of the following categories:
* **Comics**
* **Libros** (Books)
* **Manuscrito** (Handwritten Manuscripts)
* **Mecanografiado** (Typewritten Documents)
* **Tickets** (Receipts)

## 🛠 Requirements

Ensure you have the following versions installed:
* **Python 3.12**
* **OpenCV 4.11** (`opencv-python`)

## ⚙️ How to Run

Follow these steps to execute the program:

1. **Open the Terminal:** Open your terminal (CMD, PowerShell, or VS Code terminal).
2. **Navigate to the Folder:** Ensure you are in the directory where `doc_classifier.py` is saved using the `cd` command.
3. **Prepare the Image:** Have your image file (e.g., `imagen.jpg`) ready in the same folder.
4. **Execute the Program:** Run the following command in the terminal:

```bash
python doc_classifier.py imagen.jpg
```

Note: Replace imagen.jpg with the specific name of the photo you want to classify.
